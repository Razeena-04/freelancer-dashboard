# 🚀 Ultimate Freelancer Dashboard

A modern and responsive Freelancer Dashboard built using **HTML, CSS, and JavaScript**.

This project simulates a professional freelancer profile system with skill-based level detection, market demand chart visualization, and a structured roadmap for growth.

---

## 🌟 Features

- 👤 Dynamic Profile Section
- 📊 Market Demand Chart (Chart.js)
- 📈 Skill-Based Level Detection (Beginner / Intermediate / Expert)
- 🖼️ Motivation Gallery
- 📚 Freelancer Roadmap Page
- 🔁 Navigation Between Pages
- 🎨 Modern Glassmorphism UI Design
- 📱 Responsive Layout

---

## 📂 Project Structure

```

📁 Project Files
│
├── index.html       → Main input page
├── result.html      → Skill result & summary
├── dashboard.html   → Main freelancer dashboard
└── roadmap.html     → Learning roadmap page

```

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Chart.js (for market demand visualization)

---

## ⚙️ How It Works

1. User enters a skill on the main page.
2. Skill is stored using `localStorage`.
3. Dashboard dynamically:
   - Displays skill
   - Calculates experience level
   - Shows market demand chart
4. User can navigate to roadmap for learning guidance.
